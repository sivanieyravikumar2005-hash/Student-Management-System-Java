Student Management System

A Java-based Student Management System developed using Object-Oriented Programming (OOP) concepts. The application allows users to manage student records efficiently through a menu-driven interface.

Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Duplicate ID Validation
- Total Students Count
- Sort Students by ID
- Exception Handling
- File Handling for Data Persistence

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Exception Handling
- File Handling
- IntelliJ IDEA

Concepts Implemented

- Classes and Objects
- Constructors
- ArrayList Collections
- CRUD Operations
- Exception Handling
- File I/O Operations
- Sorting Techniques

Project Outcome

This project demonstrates the implementation of core Java concepts, CRUD operations, data validation, sorting, and persistent storage using file handling.
